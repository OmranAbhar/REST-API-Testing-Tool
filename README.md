# REST-API-Testing-Tool

REST API Testing Tool is a lightweight web application that runs locally in your browser. It allows developers to test REST APIs by sending HTTP requests such as GET, POST, PUT, PATCH, and DELETE. Users can add custom headers, query parameters, JSON payloads, form data, or plain text data, and view API responses in a clear format. The tool helps developers quickly verify API endpoints, debug requests, and analyze responses without requiring external software installation.
